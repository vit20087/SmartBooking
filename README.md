# Проект SmartBooking

SmartBooking — це зручна платформа для резервування послуг у різних сферах, таких як готелі, ресторани, транспорт і багато іншого. Наша мета — зробити процес бронювання максимально простим та ефективним для користувачів.

## Основні функції:
- **Легкий пошук**: Знаходьте послуги за критеріями, такими як місцезнаходження, тип послуги та ціна.
- **Користувацький інтерфейс**: Інтуїтивно зрозумілий інтерфейс для легкого навігації.
- **Безпека даних**: Захист особистої інформації користувачів.
- **Підтримка**: Наша команда підтримки завжди готова допомогти вам у випадку питань.

## Переваги використання SmartBooking:
- Економія часу
- Можливість порівняння цін
- Гнучкі умови бронювання

## Технологічний стек

### 🎨 Фронтенд:
- **React** (v19.2.4) - бібліотека для побудови користувацьких інтерфейсів
- **Vite** (v8.0.4) - швидкий інструмент збірки
- **React Router DOM** (v7.14.0) - маршрутизація сторінок
- **Axios** (v1.14.0) - HTTP клієнт для запитів до API
- **Bcryptjs** (v3.0.3) - хешування паролів

### 🔧 Бекенд:
- **Node.js** - середовище виконання JavaScript
- **Express.js** (v5.2.1) - веб-фреймворк
- **Prisma** (v5.22.0) - ORM для роботи з БД
- **SQLite** - база даних
- **bcryptjs** (v3.0.3) - хешування паролів

### 📦 Залежності:
- **CORS** (v2.8.6) - дозвіл кроз-доменних запитів
- **dotenv** (v17.4.1) - завантаження змінних оточення

## Як почати:

### Встановлення залежностей
bash
# Backend
npm install

# Frontend
cd frontend
npm install

### Запуск проекту
bash
# Запуск бекенду (слухає на порту 3000)
npm start

# У іншому терміналі - запуск фронтенду
cd frontend
npm run dev

### Ініціалізація БД
bash
# Запуск seed-скриптів для заповнення тестовими даними
node seed-masters.js
node seed-services.js
